# Public Folder

## Add Your Resume Here

To enable resume downloads on your website:

1. Save your resume PDF as: `resume.pdf`
2. Place it in this `public` folder
3. The download button will automatically work!

**Current file expected:** `resume.pdf`
**Download name shown to users:** `Gokhan_Sahin_Resume.pdf`

You can change the download filename in `src/components/Hero.tsx` if needed.
